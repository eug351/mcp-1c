# mcp-1c

MCP-сервер для интеграции AI-ассистентов (Claude, Cursor) с 1С:Предприятие.

Позволяет AI видеть метаданные вашей конфигурации 1С и генерировать точный код на BSL.

## Быстрый старт

### 1. Установить расширение в 1С

Скачайте файл расширения `MCP_HTTPService.cfe` из [Releases](../../releases)
и установите в вашу базу: Конфигуратор → Конфигурация → Расширения конфигурации → Добавить из файла.

### 2. Опубликовать HTTP-сервис

В 1С: Администрирование → Публикация на веб-сервере, или запустите базу
с параметром `/HTTPPort 8080`.

### 3. Скачать MCP-сервер

Скачайте бинарник для вашей ОС из [Releases](../../releases).

### 4. Настроить Claude Desktop

Добавьте в файл конфигурации Claude Desktop (`claude_desktop_config.json`):

```json
{
  "mcpServers": {
    "1c-enterprise": {
      "command": "C:\\path\\to\\mcp-1c.exe",
      "args": ["--base", "http://localhost:8080/mcp"],
      "env": {
        "MCP_1C_USER": "Admin",
        "MCP_1C_PASSWORD": ""
      }
    }
  }
}
```

### 5. Готово!

Откройте Claude Desktop и спросите: "Покажи структуру конфигурации моей базы 1С"

## Доступные инструменты

| Tool | Описание |
|------|---------|
| `get_metadata_tree` | Дерево метаданных: справочники, документы, регистры |
| `get_object_structure` | Реквизиты и табличные части объекта |
| `bsl_syntax_help` | Справка по встроенным функциям BSL |

## Конфигурация

| Флаг | Env var | По умолчанию | Описание |
|------|---------|-------------|----------|
| `--base` | `MCP_1C_BASE_URL` | `http://localhost:8080/mcp` | URL HTTP-сервиса 1С |
| `--user` | `MCP_1C_USER` | — | Пользователь HTTP-сервиса |
| `--password` | `MCP_1C_PASSWORD` | — | Пароль HTTP-сервиса |

Переменные окружения имеют приоритет над значениями по умолчанию, флаги CLI — над переменными окружения.
